### Публичная версия темы и плагина сайта pillars.ru

**ThePlugin** – плагин WordPress для функционального расширентия темы, такие как:
-   поля контактов и шорткоды для их вставки;
-   дополнительные колонки в админке;
-   добавление произвольных полей в посты;
-   прочее.

**Pillars** – основная тема оформления сайта, все исходные файлы для JS и CSS файлов находят в папке `html`.
